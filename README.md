# ElusiveArtFinder

A little elixir script to find the art at the end of the cool digital art experiment at [permanent-redirect](https://permanent-redirect.xyz/).

### Usage

1. git clone && cd
2. `mix deps.get`
3. `iex -S mix`
4. `ElusiveArtFinder.fetch_page`
5. aaannnddd.. wait :D

### Dependencies

* HTTPoison
